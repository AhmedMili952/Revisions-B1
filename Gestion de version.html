QCM XTI103 – Git & GitHub

1. Quelle commande permet de créer une branche dev ?

git switch dev
git branch dev
git merge dev
git pull dev

2. Qu’est-ce qu’une branche Git ?

Une copie complète du dépôt
Une ligne de développement indépendante
Un simple dossier
Un commit spécial

3. Qu’est-ce qu’un commit Git ?

Une modification temporaire
Une sauvegarde de l’état du projet
Une branche distante
Un dépôt GitHub

4. Que fait la commande git merge ?

Supprime une branche
Intègre une branche dans une autre
Renomme une branche
Annule un commit

5. Que représente HEAD dans Git ?

Le premier commit
Le commit parent du prochain commit
Le dépôt distant
Un tag GitHub
